📋 部署步骤

方法一：GitHub 自动部署（推荐）

  1. 推送代码到 GitHub: git push origin main
  2. 登录 netlify.com
  3. 选择 "New site from Git" → GitHub → 选择仓库
  4. 构建设置：Branch: main, Build command: echo 'No build needed', Publish directory: .
  5. 点击 "Deploy site"

方法二：拖拽部署

  1. 确保所有文件在项目根目录
  2. 登录 Netlify → "Deploy manually"
  3. 拖拽整个项目文件夹到部署区域
